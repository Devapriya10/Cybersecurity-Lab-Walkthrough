# Cybersecurity-Lab-Walkthrough

## Overview
This project demonstrates a penetration testing walkthrough performed in a controlled lab environment.

## Activities Performed
- Network scanning
- Service enumeration
- Directory brute forcing
- LFI testing
- SSH access
- Privilege escalation

## Tools Used
- Kali Linux
- Nmap
- Metasploit Framework
- Hydra
- Dirsearch
- Wireshark

## Disclaimer
This project was conducted in an authorized lab environment strictly for educational and ethical cybersecurity research purposes.
